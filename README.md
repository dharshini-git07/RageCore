# 🎮 RageCore – Gaming Store Website

RageCore is a modern and responsive Gaming Store website developed as a team project using HTML, CSS, and JavaScript. It provides an engaging platform for gamers to explore popular titles, view game details, play mini browser games, and enjoy a premium gaming-inspired interface.

## 🚀 Features

- Responsive and modern UI
- Hero Slider
- Featured Games
- Trending Games
- Game Catalog
- Game Details Page
- Search Functionality
- Category & Price Filters
- Mini Games
  - 🚗 Dash Cars
  - 🐍 Snake
  - ❌ Tic Tac Toe
  - 🃏 Memory Card Game
  - ✊✋✌️ Rock Paper Scissors
- Special Offer Banner with Countdown Timer
- Why Choose RageCore Section
- Newsletter Subscription
- Responsive Navigation Bar
- Mobile Sidebar
- Footer with Quick Links

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome
- JSON (Game Data)
- Git & GitHub
